Implementar Flask em Python

Instalação do Flask

O Flask fornece configuração e convenções com padrões sensatos para começar. Esta seção da documentação explica as diferentes partes da estrutura do Flask e como elas podem ser usadas, personalizadas e estendidas. Além do Flask em si, você pode procurar por extensões mantidas pela comunidade para adicionar ainda mais funcionalidades.

Para começar, crie uma pasta de projeto e dentro dela crie uma pasta chamada 'venv' com o seguinte comando: py -3 -m venv venv

Em seguida, ative o ambiente virtual com o comando: venv\Scripts\activate

Por fim, instale o Flask com o comando: pip install Flask
