Iniciando em Banco de Dados

Instalação

Preparar um banco de dados não é uma tarefa complicada, tendo como base os requisitos de utilizar o Flask e HTML para criar uma máquina virtual.

Como instalar o Flask

Após atender os requisitos, execute o seguinte comando:

pip install SQLAlchemy

Com o SQLAlchemy instalado, você pode começar a usá-lo para interagir com bancos de dados relacionais. Aqui está um pequeno exemplo de como criar uma tabela usando o SQLAlchemy:

Agora digite os seguintes comandos em seu terminal:

  • flask db init
  • flask db migrate -m "db_criar_banco"
  • flask db upgrade

Isso ativará seu banco de dados e permitirá que você cadastre informações nele.
